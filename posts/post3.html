Tip Calculator Web Application

This is a simple web application that takes user input to return the proper amount of tip to return. It stores the data of the last usage to let the user know a little about their hitsory using the html localStorage functionality

The application the just gets this information and prints it to the screen upon your return.

The core of the application is very simple, it just multiples the inputed bill by the inputed percent that is converted to decimal and then returns the result.

The Final addition I made to this Application was adding a graphics the is somwhat similar to an icon in the top left hand corner using canvas.

I chose colors similar to my portfolio page and added two slighty transparent overlapping boxes as I saw in a tutorial and liked how to appeared.
